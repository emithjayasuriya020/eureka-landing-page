# Eureka - Institutional Precision in Recovery

> The next generation of campus lost and found. A centralized, secure, and AI-driven ecosystem designed exclusively for University Students.

---

## 🌟 Overview

Eureka is an advanced platform designed to streamline the process of recovering lost items on university campuses. By leveraging AI-driven matching and secure communication protocols, Eureka ensures that lost property is returned to its rightful owner with institutional precision and maximum security.

## 🚀 Key Features

- **🤖 AI Smart Matching**: Sophisticated neural networks compare lost reports with found items in real-time, delivering high-probability matches instantly.
- **🛡️ Fraud Prevention**: Public lists of found items are obscured. Only potential owners see relevant identifying details after initial verification.
- **💬 Secure Chat Verification**: Encrypted, anonymous communication channels allow users to verify ownership before physical recovery.
- **✅ Trust & Safety**: Integrated protocols to ensure a safe and reliable environment for all campus community members.

## 🛠️ Technical Stack

- **Frontend**: [React 19](https://react.dev/), [Vite](https://vitejs.dev/), [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Communication**: [EmailJS](https://www.emailjs.com/)
- **Icons**: Custom SVG & Lucide-like iconography

## ⚙️ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- [npm](https://www.npmjs.com/)

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/emithjayasuriya020/eureka-landing-page.git
   cd eureka-landing-page
   ```

2. **Install root dependencies**:
   ```bash
   npm install
   ```

3. **Install React app dependencies**:
   ```bash
   cd react
   npm install
   ```

### Running the Project

1. **Start the development server**:
   ```bash
   npm run dev
   ```
2. Open your browser and navigate to `http://localhost:5173`.
